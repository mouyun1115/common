# common

## 简介

common是使用C++11编写的一系列基础的业务模块，主要实现了流水线架构的基础设施，同时基于C++11完美支持跨平台

## 设施列表
```
- common_header     一些通用宏和函数的定义
- AnyType           弱类型，类似于C#的var
- Buffer            字节数据存取
- IoC               依赖反转的C++实现
- Timer             高精度计时器
- Event             事件
- Semphore          信号量
- ObjectPool        对象池
- Processor         流水线单元
```
